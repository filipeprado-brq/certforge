```typescript
export const STORAGE_KEY = 'cae-trainer:v1';
export const SCHEMA_VERSION = 1;
export type ThemePref = 'system' | 'light' | 'dark';
export interface PersistedState {
  schemaVersion: number;
  themePref: ThemePref;
  // later phases extend this (srs schedule, quiz history) — keep additive
}
export function readState(): PersistedState;        // returns defaults if absent/invalid
export function writeState(next: PersistedState): void;
export function patchState(patch: Partial<PersistedState>): PersistedState; // read-merge-write
export function clearAll(): void;                   // removes the cae-trainer:v1 namespace
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| app → localStorage | The only persistence sink; values are produced and consumed solely by this app on the same origin. No server, no other origin. |
| build → static host | `dist/` served as static files; no server-side execution. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | localStorage (`cae-trainer:v1`) | mitigate | readState validates `schemaVersion` and falls back to defaults on mismatch/corruption; all parsing wrapped in try/catch so tampered/garbage values never crash the app |
| T-01-02 | Denial of Service | localStorage write (quota / disabled storage) | mitigate | writeState/readState wrapped in try/catch; failures degrade to in-memory defaults rather than throwing |
| T-01-03 | Information Disclosure | persisted study state | accept | Data is non-sensitive study progress on the user's own device; offline-first by design (PROJECT.md), no PII, no network transmission |
| T-01-04 | Tampering | bundled static assets | accept | Static offline app with no integrity-critical secrets; host-level integrity is out of v1 scope |
</threat_model>
